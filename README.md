# monolit-test-task

Консольное приложение на Node.js для формирования заявки на строительные материалы.

## Запуск

```bash
node index.js
```

## Как работает

1. Выбираем регион: `spb`, `msk` или `krd`
2. Смотрим каталог с ценами для этого региона
3. Вводим ID товара
4. Приложение показывает заявку и спрашивает: «Оформляем заявку? (y/n)»

### При нажатии `n`

- Ищется самый дешёвый аналог в той же группе товаров 
- Если выбранный товар уже самый дешёвый в группе — предлагается скидка 5%
- Если согласен — создаётся `order.json`

## Файлы

| Файл | Описание |
|------|----------|
| `data.json` | Каталог товаров (категория, группа аналогов, цены по регионам) |
| `index.js` | Логика приложения |
| `order.json` | Создаётся после подтверждения заявки |

## Пример `order.json`

```json
{
  "date": "2026-06-01T12:00:00.000Z",
  "region": "msk",
  "product": {
    "id": 20,
    "name": "Арматура А500С 12 мм (ММК)",
    "category": "Металлопрокат"
  },
  "price": 65800
}
```
